API Documentation

ClawMart's API is designed for AI agents. No authentication required for read endpoints. All responses are JSON with a consistent schema.

Base URL: {{ request.url_root.rstrip('/') }}
All read endpoints are open — no API key, no tokens, no sign-up. Just send requests and get JSON back.
GET /api/v1/servers
List and search MCP servers in the registry. Supports text search, category filtering, and tag filtering.
Query Parameters
NameTypeDescription
searchstringSearch in name, description, and tags
categorystringFilter by category (e.g. "Developer Tools")
tagsstringComma-separated tags to filter by
limitintMax results (default: 50, max: 200)
Example Request
curl {{ request.url_root.rstrip('/') }}/api/v1/servers?search=database&limit=5
Example Response
{
  "status": "success",
  "data": {
    "servers": [{
      "id": 5,
      "name": "postgres",
      "description": "PostgreSQL database integration...",
      "capabilities": {"tools": ["query"]},
      "install_command": "npx -y @modelcontextprotocol/server-postgres"
    }],
    "total": 1
  }
}
GET /api/v1/servers/{id}
Get full details for a single MCP server by its ID.
Example
curl {{ request.url_root.rstrip('/') }}/api/v1/servers/1
POST /api/v1/discover
The killer feature. Describe what you need in plain English and ClawMart uses Claude to find the most relevant MCP servers. Perfect for agents that know what they need but not which server provides it.
Request Body (JSON)
NameTypeDescription
capabilitystringNatural language description of what you need (required, min 5 chars)
limitintMax matches (default: 5, max: 10)
Example Request
curl -X POST {{ request.url_root.rstrip('/') }}/api/v1/discover \
  -H "Content-Type: application/json" \
  -d '{"capability": "I need to read and write files on the local system"}'
Example Response
{
  "status": "success",
  "data": {
    "query": "I need to read and write files on the local system",
    "matches": [{
      "server": {"id": 1, "name": "filesystem", ...},
      "relevance_score": 10,
      "reason": "Provides read_file, write_file, and directory management tools"
    }],
    "count": 1
  },
  "metadata": {"powered_by": "Claude (Anthropic)"}
}
GET /api/v1/categories
List all categories with the number of servers in each.
Example
curl {{ request.url_root.rstrip('/') }}/api/v1/categories
